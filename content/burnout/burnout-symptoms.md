このような状態はバーンアウト（燃え尽き症候群）のサインかもしれません。バーンアウトは真面目で熱心に働く人ほど陥りやすく、放置すると回復に時間がかかります。

この記事では、**段階別のチェックリストで自分の状態を確認**しながら、バーンアウトの正体と回復法を解説します。

バーンアウトは「怠け」や「意志の弱さ」ではありません。長期間にわたって頑張り続けた結果、心と体の資源が枯渇した状態です。真面目で責任感が強く、仕事に情熱を持って取り組んできた人ほどリスクが高いと言われています。

function boCheck(el) {
  el.classList.toggle('checked');
  var section = el.closest('.bo-list');
  var id = section.id;
  var count = section.querySelectorAll('.checked').length;
  var countEl = document.getElementById(id.replace('bo-', 'bo-') + '-count');
  if (countEl) countEl.textContent = count;
}

function boJudge() {
  var early = document.getElementById('bo-early').querySelectorAll('.checked').length;
  var mid = document.getElementById('bo-mid').querySelectorAll('.checked').length;
  var severe = document.getElementById('bo-severe').querySelectorAll('.checked').length;
  var total = early + mid + severe;
  var result = document.getElementById('bo-result');

  var bg, border, color, judge, desc;
  if (severe >= 2) {
    bg = '#FEF2F2'; border = '#EF4444'; color = '#B91C1C';
    judge = '重症期のサインが複数あります。今すぐ専門家に相談してください';
    desc = '重症サインが複数当てはまっています。これは意志の弱さや甘えではありません。心身が限界を超えているサインです。今すぐ心療内科・精神科への相談、または職場の産業医への相談を検討してください。一人で抱え込まないでください。';
  } else if (severe >= 1 || mid >= 4) {
    bg = '#FFE4E6'; border = '#FCA5A5'; color = '#9F1239';
    judge = '消耗が進んでいます。休息と専門家への相談を検討してください';
    desc = '中期〜重症期のサインが見られます。今の状態を放置すると、さらに悪化する可能性があります。まず休息を最優先にし、心療内科や産業医への相談も検討してください。休職も選択肢の一つです。';
  } else if (mid >= 2 || early >= 4) {
    bg = '#FEF9C3'; border = '#FDE047'; color = '#854D0E';
    judge = '疲れが蓄積しています。早めのケアが大切です';
    desc = '初期〜中期のサインが見られます。今はまだ自分でケアできる段階ですが、放置すると悪化します。休日に仕事から完全に離れる時間を作る、睡眠を優先する、信頼できる人に話すなど、早めのケアを始めましょう。';
  } else if (total >= 1) {
    bg = '#DBEAFE'; border = '#93C5FD'; color = '#1E40AF';

    bg = '#DCFCE7'; border = '#86EFAC'; color = '#166534';
    judge = '現時点では大きなサインは見られません';
    desc = 'チェック項目に当てはまるものが少ない状態です。ただしバーンアウトは徐々に進行するため、定期的に自分の状態を確認する習慣を持ちましょう。';
  }

  result.className = 'bo-result show';
  result.style.background = bg;

#### STEP 1. 徹底的に休む

最も重要なのは心身を徹底的に休ませることです。「休んでも罪悪感がある」という人は多いですが、バーンアウトの回復には休息が不可欠です。仕事から完全に離れ、十分な睡眠・栄養バランスの良い食事・ゆっくりできる時間を確保しましょう。重症の場合は休職も有効な選択肢です。

#### STEP 2. ストレス源から距離を置く

バーンアウトの原因となっているストレス源を特定し、距離を置きましょう。過剰な業務量・人間関係のトラブル・職場環境のミスマッチなど、原因は人によって異なります。業務内容の変更・部署異動・テレワークの活用など、環境を調整することで心身の回復が早まることがあります。

#### STEP 3. 専門家のサポートを受ける

一人で抱え込まず、心療内科・精神科の医師・産業医・カウンセラーなどの専門家に相談しましょう。「病院に行くほどではないかも」と感じる段階でも、早めに相談することで症状の悪化を防げます。専門家は適切な診断と回復への道筋を示してくれます。

#### STEP 4. 焦らず段階的に復帰する

体力と気力が戻ってきたら、無理のない範囲で少しずつ活動量を増やしていきましょう。「早く元に戻らなければ」と焦ることがバーンアウトの再発につながります。完全回復を急がず、自分のペースで進めることが大切です。

ブラック企業あるある。笑えるけど笑えない、あなたの会社は大丈夫？

ブラック企業の特徴は、一見すると「どこにでもあること」のように感じられる点にあります。しかし、その「あるある」が積み重なることで、あなたの心身は確実に疲弊していきます。強いストレス状態が続くと、心身の消耗が続いている状態のことがあります。この「笑えるけど笑えない」という絶妙なトーンの「あるある」を通じて、あなたの会社が本当に健全な環境なのか、客観的に見つめ直すきっかけにしてください。

「共感エンタメ」で気づく、あなたの会社のブラック度

ブラック企業あるあるは、まさに「共感エンタメ」です。多くの人が経験しているであろう「あるある」に触れることで、自分だけではないという安心感とともに、自分の置かれている状況の異常性に気づくことができます。感覚が麻痺してしまっていると、客観的な判断が難しくなります。このあるあるリストで、あなたの会社のブラック度をチェックし、現状を整理してみましょう。

あなたの会社に当てはまるものはいくつありますか？

労働時間あるある

  1. 定時で帰ろうとすると「やる気がない」と言われる
    • 残業が評価の基準になっている証拠です。
  2. 休憩時間がまともに取れない
    • ランチはデスクで、トイレに行くのも一苦労…なんてことありませんか？
  3. 休日出勤は当たり前、代休は取れない
    • 「みんなやってるから」という魔法の言葉で片付けられがちです。
  4. 終電を逃すのは日常茶飯事、タクシー代は自腹
    • 会社は社員の生活を顧みているでしょうか。

給与あるある

  1. 「みなし残業」でいくら残業しても給料が変わらない
    • サービス残業の温床になりやすい制度です。
  2. 基本給が異常に低く、手当でかさ増しされている
    • ボーナスや退職金計算に影響が出る可能性があります。
  3. ボーナスは寸志、または支給されない
    • 業績が悪いから、と毎年同じ説明が繰り返されます。
  4. 給与明細を見ても何が何だか分からない
    • 不透明な給与体系は、不信感の元です。

人間関係あるある

  1. 上司の機嫌で職場の空気が一変する
    • 常に顔色を伺う必要があり、精神的に疲弊します。
  2. パワハラ・モラハラが日常的に行われている
    • 「指導」という名のいじめが横行していませんか？
  3. 社員同士の足の引っ張り合いが激しい
    • チームワークよりも個人間の競争が煽られています。
  4. 退職者が後を絶たないのに、誰も問題視しない
    • 人の入れ替わりが激しいのは、環境に問題がある証拠です。

採用・教育あるある

  1. 求人情報と実際の業務内容が全く違う
    • 入社後に「話が違う」と感じることはありませんか？
  2. 研修はOJT任せで、まともな教育制度がない
    • 新人が育ちにくい環境は、会社の成長を阻害します。
  3. 「うちは家族みたいな会社だから」とプライベートに踏み込んでくる
    • 公私の区別が曖昧な職場は要注意です。
  4. 入社してすぐに「辞めたい」と言い出す人がいる
    • 会社の雰囲気に耐えられない人が多い証拠です。

組織文化あるある

  1. 飲み会や社内イベントへの参加が強制される
    • プライベートな時間を会社に拘束されるのは苦痛です。
  2. 精神論や根性論がまかり通っている
    • 具体的な改善策よりも、気合で乗り切ろうとします。
  3. 「お客様は神様」を履き違え、過剰なサービスを強要される
    • 社員の犠牲の上に成り立つサービスは持続可能ではありません。
  4. 会社の未来について語る人がいない
    • 社員が会社の将来に希望を持てないのは、深刻な問題です。
